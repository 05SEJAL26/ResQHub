# ResQHub – Emergency Resource Allocation System

A Java-based desktop application for managing and tracking emergency medical resources across multiple hospitals.

## Features

* Role-based access control (Admin/User)
* Hospital resource management
* Search, filter, and sort functionality
* Emergency resource request management
* Low-resource alert system
* MySQL database integration

## Technologies Used

* Java
* Java Swing
* MySQL
* JDBC
* HashMap
* Object-Oriented Programming (OOP)
